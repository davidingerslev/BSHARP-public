# Bristol Supported Housing Analysis Research Project

This is the source code for analysis completed as part of research into supported housing in Bristol.

## Getting Started

The best place to start is the Quarto generated website: [https://davidingerslev.github.io/BSHARP-public/](https://davidingerslev.github.io/BSHARP-public/)

## Data cleaning

The data cleaning and filtering work is done by the Python script library -- see the [python_scripts](python_scripts) folder.

## Output

The output from the analysis (figures and tables) is in the [assets_output](assets_output) folder.
